This is a basic front-end microproject developed using HTML and CSS.  
The project demonstrates structured webpage layout, styling, and clean UI implementation.

## How to Access the Website

1. Go to this GitHub repository.
2. Click the **Code** button.
3. Select **Download ZIP**.
4. Extract the downloaded folder.
5. Open the extracted folder.
6. Double-click `index.html` to open the website in your browser.

No additional setup is required since this is a static HTML/CSS project.
